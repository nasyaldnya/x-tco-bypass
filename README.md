## 🛠️ Installation / طريقة التثبيت / インストール / 설치

### 🇬🇧 English

**Method 1: Automatic Install (Recommended)**
1. Install a user script manager like [Tampermonkey](https://www.tampermonkey.net/).
2. Click on the `tco-bypass.user.js` file in this repository.
3. Click the **Raw** button at the top right of the file view.
4. Tampermonkey will ask you to install the script. Click **Install**.
5. Refresh X/Twitter and enjoy!

**Method 2: Manual Install**
1. Open the Tampermonkey Dashboard and click the **+** (Create a new script) tab.
2. Copy the entire code from the `tco-bypass.user.js` file.
3. Paste it into the Tampermonkey editor (replacing all default code).
4. Go to **File > Save** (or press `Ctrl+S`).

### 🇸🇦 العربية

**الطريقة الأولى: التثبيت التلقائي (موصى به)**
1. قم بتثبيت إضافة إدارة السكربتات مثل [Tampermonkey](https://www.tampermonkey.net/).
2. اضغط على ملف `tco-bypass.user.js` الموجود في هذه الصفحة.
3. اضغط على زر **Raw** في الزاوية العلوية اليمنى للكود.
4. ستفتح لك صفحة التثبيت الخاصة بـ Tampermonkey، اضغط على **تثبيت (Install)**.
5. قم بتحديث صفحة إكس / تويتر!

**الطريقة الثانية: التثبيت اليدوي**
1. افتح لوحة تحكم (Dashboard) إضافة Tampermonkey، واضغط على أيقونة **+** (إنشاء سكربت جديد).
2. انسخ الكود بالكامل من ملف `tco-bypass.user.js`.
3. الصقه في المحرر (مع مسح الأكواد الافتراضية الموجودة مسبقاً بالكامل).
4. احفظ السكربت من **File > Save** (أو اضغط `Ctrl+S`).

### 🇯🇵 日本語

**方法 1: 自動インストール (推奨)**
1. [Tampermonkey](https://www.tampermonkey.net/) などのユーザースクリプトマネージャーをインストールします。
2. このリポジトリにある `tco-bypass.user.js` ファイルをクリックします。
3. ファイルビューの右上にある **Raw** ボタンをクリックします。
4. Tampermonkey のインストール画面が開くので、**インストール (Install)** をクリックします。
5. X（Twitter）のページを更新（リロード）して完了です！

**方法 2: 手動インストール**
1. Tampermonkeyのダッシュボード開き、**+**（新規スクリプトを追加）タブをクリックします。
2. `tco-bypass.user.js` ファイルからコード全体をコピーします。
3. エディタに貼り付けます（デフォルトのコードはすべて消去してください）。
4. **ファイル > 保存** をクリックします（または `Ctrl+S` を押します）。

### 🇰🇷 한국어

**방법 1: 자동 설치 (권장)**
1. [Tampermonkey](https://www.tampermonkey.net/)와 같은 사용자 스크립트 관리자를 설치합니다.
2. 이 저장소에 있는 `tco-bypass.user.js` 파일을 클릭합니다.
3. 파일 보기의 오른쪽 상단에 있는 **Raw** 버튼을 클릭합니다.
4. Tampermonkey 설치 화면이 나타나면 **설치 (Install)**를 클릭합니다.
5. X(Twitter) 페이지를 새로 고침하면 완료됩니다!

**방법 2: 수동 설치**
1. Tampermonkey 대시보드를 열고 **+** (새 스크립트 만들기) 탭을 클릭합니다.
2. `tco-bypass.user.js` 파일에서 전체 코드를 복사합니다.
3. 편집기에 붙여넣습니다 (기본 코드는 모두 지웁니다).
4. **파일 > 저장**을 클릭합니다 (또는 `Ctrl+S`를 누릅니다).
